# Numerical-Conversion
Converting numbers. (Binary, Hexa and Octal)

           Sobre:
Ele foi criado para auxiliar nas atividades do meu curso sobre conversões numéricas. Além de me ajudar nas tarefas, também me ajudou a compreender melhor a lógica das funções.
___
           
           Conceitos aplicados:
   
-Sistemas de numeração.
    
-Conversão entre bases numéricas
    
-Lógica de programação
    
-Estruturas básicas da linguagem

          Funcionalidades:
   
Converter binário → decimal

Converter decimal → binário

Converter decimal → hexadecimal

Converter decimal → octal

Conversões entre outros sistemas numéricos

Loop para repetir o processo
____
           Objetivo:

Feito com o objetivo de praticar a minha lógica e conhecimento com estruturas aninhadas, funções, indentação e condições.


    Como executar o projeto:

1• Clone esse repositório: https://github.com/Epic10ka/Numerical-Conversion

2• Acesse a pasta do projeto 

3• Execute o arquivo `nums_conversions.py` e teste-o livremente.
